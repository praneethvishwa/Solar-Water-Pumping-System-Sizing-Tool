# Solar-Water-Pumping-System-Sizing-Tool
The Solar Water Pumping System (SWPS) Sizing Tool is a python-based engineering application developed to simplify the design and sizing of solar-powered irrigation systems.
application developed to simplify the design and sizing of solar-powered irrigation systems. The tool estimates the required photovoltaic (PV) array capacity, pump power, daily water demand, irrigation requirements, and system specifications based on user inputs such as crop type, cultivated area, climatic conditions, irrigation method, and pumping head.
The sizing calculations are based on established agricultural engineering principles, including the FAO Penman–Monteith method for reference evapotranspiration (ET₀), crop coefficient (Kc) values for crop water requirement (ETc), and hydraulic power calculations for pump sizing. The tool enables farmers, researchers, and engineers to design an efficient and sustainable solar pumping system without performing complex manual calculations.

Key Features

Calculates crop water requirement (ETc) using ET₀ and Kc values.
Estimates daily irrigation water demand.
Sizes the required solar PV array and water pump.
Considers total dynamic head (TDH) and irrigation efficiency.
Provides a user-friendly web interface for quick system design.
Reduces design time while improving sizing accuracy.

- Pump sizing calculations
- Solar PV sizing
- Hydraulic head calculations
- Performance optimization
- Irrigation system analysis
- Excel input/output support
